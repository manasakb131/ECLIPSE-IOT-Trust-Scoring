# 🚀 Dynamic Trust Scoring for Secure IoT Networks

### 🔐 ECLIPSE-IoT | Team Vision Vortex

---

## 📌 Overview

IoT devices are rapidly growing in industries such as healthcare, manufacturing, and smart infrastructure. However, these devices often lack proper security monitoring, making them vulnerable to attacks.

This project presents a **behavior-based IoT security system** that continuously monitors device activity and assigns a **dynamic trust score** to detect anomalies and behavioral drift.

---

## ❗ Problem Statement

* Lack of continuous monitoring for IoT devices
* High false positives in existing systems
* No clear explanation for detected risks
* Failure to detect **behavioral drift** (slow changes over time)

👉 This makes it difficult for security teams to identify real threats.

---

## 💡 Proposed Solution

**ECLIPSE-IoT** is designed to:

* Monitor IoT devices using **network telemetry (PCAP/CSV)**
* Learn normal behavior of devices
* Detect **anomalies and behavioral drift**
* Assign a **Trust Score (0–100)**
* Provide **clear and explainable alerts**

---

## 🏗️ System Architecture

```
[ Network Data (PCAP / CSV) ]
            ↓
[ Feature Extraction ]
            ↓
[ ML Model (Anomaly Detection + Drift Analysis) ]
            ↓
[ Policy Engine ]
            ↓
[ Trust Score Engine ]
            ↓
[ Alerts & Visualization Dashboard ]
```

---

## ⚙️ Workflow / Pipeline

1. Collect network telemetry data
2. Learn baseline (normal behavior)
3. Monitor real-time device activity
4. Detect anomalies & behavioral drift
5. Apply policy rules
6. Update trust score dynamically
7. Generate alerts with explanation

---

## ✨ Key Features

* 🔢 Dynamic Trust Score (0–100)
